# Tondix CRM Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adapter Atomic CRM en CRM Tondix avec catalogue produits/services, liaisons deal↔catalogue, contrats d'entretien automatiques, et widget de relance.

**Architecture:** Cinq nouvelles tables Supabase (products, services, deal_products, deal_services, maintenance_contracts) + trigger automatique de création de contrat lors du passage d'un deal en "won". Côté frontend, de nouvelles pages catalogue sous `/catalogue/*`, des modifications du formulaire deal avec `useFieldArray`, et de nouveaux composants dans la fiche entreprise et le dashboard.

**Tech Stack:** React 19, TypeScript strict, React Admin (ra-core), Supabase (PostgreSQL + RLS), Tailwind CSS v4, shadcn/ui, Vitest + Testing Library.

- [ ] **Étape 2 : Ajouter la fonction trigger dans 02_functions.sql**

Ajouter à la fin de `supabase/schemas/02_functions.sql` (avant la dernière ligne si applicable) :

```sql
-- Create maintenance contracts when a deal is won
create or replace function public.create_maintenance_contracts_on_win()
returns trigger
language plpgsql
security definer
as $$
begin
    -- Only trigger when stage changes to 'won'
    if (old.stage is distinct from new.stage and new.stage = 'won') then
        insert into public.maintenance_contracts (company_id, service_id, deal_id, start_date, end_date)
        select
            new.company_id,
            ds.service_id,
            new.id,
            current_date,
            current_date + (s.periodicity_months || ' months')::interval
        from public.deal_services ds
        join public.services s on s.id = ds.service_id
        where ds.deal_id = new.id;
    end if;
    return new;
end;
$$;
```

```sql
select

    c.size,
    c.linkedin_url,
    c.website,
    c.phone_number,

    c.country,
    c.description,
    c.revenue,
    c.tax_identifier,
    c.logo,
    count(distinct d.id) as nb_deals,
    count(distinct co.id) as nb_contacts,
    exists(
        select 1 from public.deal_products dp
        join public.deals d2 on d2.id = dp.deal_id
        where d2.company_id = c.id
    ) as has_equipment,
    exists(
        select 1 from public.maintenance_contracts mc
        where mc.company_id = c.id and mc.end_date > now()
    ) as has_maintenance
from public.companies c
    left join public.deals d on c.id = d.company_id
    left join public.contacts co on c.id = co.company_id
group by c.id;
```

```typescript
import type { DataProvider } from 'ra-core';
import type { DealProductLine, DealServiceLine } from '../types';

export async function syncDealProducts(
  dataProvider: DataProvider,
  dealId: number | string,
  lines: DealProductLine[],
) {
  // Delete all existing deal_products for this deal
  const existing = await dataProvider.getList('deal_products', {
    filter: { deal_id: dealId },
    pagination: { page: 1, perPage: 1000 },
    sort: { field: 'id', order: 'ASC' },
  });
  await Promise.all(
    existing.data.map((item) =>
      dataProvider.delete('deal_products', { id: item.id, previousData: item }),
    ),
  );
  // Create new lines
  await Promise.all(
    lines
      .filter((l) => l.product_id)
      .map((line) =>
        dataProvider.create('deal_products', {
          data: {
            deal_id: dealId,
            product_id: line.product_id,
            quantity: line.quantity,
            unit_price: line.unit_price,
          },
        }),
      ),
  );
}
```
